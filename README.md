# Extended from [create-react-app](https://create-react-app.dev/)  with [craco](https://craco.js.org/)
Every time when refreshing the page, the data  would be re-generated.

## Install, using [pnpm](https://pnpm.io/)
```shell
pnpm install
```
## Start
```shell
pnpm start
```
## Build
```shell
pnpm build
```
